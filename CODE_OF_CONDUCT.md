# Code of Conduct

Участники проекта обязуются общаться уважительно и по существу.

Недопустимы:

- оскорбления, угрозы и дискриминация;
- публикация чужих персональных данных или секретов;
- намеренное добавление вредоносного кода;
- спам и вводящие в заблуждение отчёты.

Нарушения можно сообщить владельцу репозитория через профиль [@hikkiway](https://github.com/hikkiway).
